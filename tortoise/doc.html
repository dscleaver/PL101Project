Tortoise

Tortoise is an imperative language designed for drawing graphics by controling a turtle (the cursor) on a screen. The turtle always begins in the center of the canvas with the pen down.

Tortoise Evaluation

Tortoise programs consist of a series of statements containing expressions. The result of a series of statements is the result of the last statement executed.

Statements

Variable Declaration var <identifier>; or var <identifier> := <expression>;
Declare a variable in the current environment. The first form declares a variable with the default value of 0. The second assigns the result of the expression to the variable. Throws an error if the variable has already been defined within the environment.
Variable Assignment <identifier> := <expression>;
Assigns a value to a previously declared variable. Throws an error if the variable has not been defined.
Function Definition define <identifier>() { <statements> } or define < identifier>(<comma separated list of identifier>) { <statements> }
Defines a function in the current environment if the name does not already exist.
Conditional Execution if (<expression>) { <statements> }
Executes the statements if the expression returns a true value.
Repeated Execution repeat (<expression>) { <statements> }
Repeats the statements n times where n is the result of evaluating the expression.
Expression Statements <expression>;
Evaluates the expression and returns it's value.

Expressions

Numbers
Numbers evaluate to themselves.
Arithmetic Operations <expression> <op> <expression>
Tortoise supports +, -, *, and /.
Comparison Operations <expression> <op> <expression>
Tortoise supports <, >, <=, >=, ==, and !=
Variables <identifier>
Accesses the value assigned to a variable within the current environment.
Function calls <identifier>() or <identifier>(<comma separated list of expression>)
Looks up the function within the current environment and calls it with the results of the arguments

Built-in Functions

forward(n)
Moves the turtle forward n pixels
left(angle)
Rotates the turtle angle degrees to the left
right(angle)
Rotates the turtle angle degrees to the right
penup()
Lifts the pen so the turtle won't draw.
pendown()
Lowers the pen so the turtle will draw.
